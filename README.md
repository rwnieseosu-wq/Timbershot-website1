# TimberShop Multi-Page

A premium, beginner-focused woodworking website built with Next.js and Tailwind CSS.

## Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React

## Site structure
- `/` Home
- `/projects`
- `/tools`
- `/materials`
- `/workshop`
- `/resources`

## Run locally
```bash
npm install
npm run dev
```

## Build for production
```bash
npm run build
npm run start
```

## Deploy to Vercel
1. Create a new Git repository and push this project.
2. Import the repository into Vercel.
3. Accept the default framework detection for Next.js.
4. Deploy.

## Notes
- The email signup is currently visual/demo only.
- Navigation now uses separate routes rather than one scrolling page.
